ImgTor (maintained fork)

Extensible image editing tool powered by HTML5 canvas.

Introduction

ImgTor is a JavaScript library which provides basic image editing tools in your browser, such as rotation, cropping, filters, finetune (brightness, contrast, …), resize, frames, fill, redact, annotate, decorate, and retouch hooks. It is built on the HTML5 canvas API (Canvas 2D).

Why?

It's easy to get a javascript script to crop an image in a web page. ImgTor bundles common canvas workflows in plugins you can enable or disable per instance.

The concept

The core turns the target image into a canvas surface and toolbar. Features live in plugins under lib/js/plugins/ (flat files or namespaced folders).

Features

This demo enables every bundled plugin so you can try them from the toolbar:

History
Undo / redo transformations.
Rotate
Rotate left or right.
Crop
Drag a crop zone (quick key C when configured).
Filter
Colour matrix presets (Classic, Monochrome, Sepia, …).
Finetune
Open the panel for brightness, contrast, saturation, exposure, gamma, clarity, vignette.
Resize
Preset sizes or width/height inputs, then Apply.
Frame
Pick a frame style, then Bake frame.
Fill
Swatch background preview, then Bake fill.
Redact
Toggle draw, drag a region, Apply redactions.
Annotate
Rect tool in image space; Bake annotations.
Decorate
Rect in viewport space; Bake decorate.
Retouch
Select region; host can wire onSelectionComplete.
Save
Default: replaces the editor with a flattened image.
